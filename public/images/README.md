# Images Required for Evol Jewels

## Customise Page Images

### Mosaic Images (Hero Section)

Place these images in `public/images/customise/`:

- `mosaic-1.jpg` - Bespoke craftsmanship image (artisan working)
- `mosaic-2.jpg` - Artisan at work (close-up of hands crafting)
- `mosaic-3.jpg` - Custom jewellery design (sketches or 3D renders)
- `mosaic-4.jpg` - Finished masterpiece (completed custom piece)

**Recommended specs:** 1200x1600px, landscape orientation, high quality

### Panel Images (Bespoke Process Section)

Place these images in `public/images/customise/`:

- `panel-1.jpg` - Bespoke forms (detailed craftsmanship, precision tools)
- `panel-2.jpg` - A note from the heart (customer consultation, design sketches)
- `panel-3.jpg` - Thorough process (workshop, materials, crafting stages)

**Recommended specs:** 1600x1200px, high quality, full-bleed

## Journal Images

Place these images in `public/images/journal/`:

- `signet-rings.jpg` - The Return of Signet Rings article image
- `asymmetric-engagement.jpg` - Why Asymmetric Engagement Rings Matter article image

## Notes

- All images should be optimized for web (compress without losing quality)
- Use WebP format where possible for better performance
- Ensure proper licensing/ownership of all images
- Consider brand colors: Evol Red (#9F0B10), Dark Grey (#2D2D2D), Metallic (#6B6B6B)
